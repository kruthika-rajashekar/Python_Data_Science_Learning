# 📊 Python Data Analysis Project

## Libraries Used

* **Pandas** – Data loading \& cleaning
* **NumPy** – Numerical operations
* **Matplotlib** – Data visualization
* **Scikit-Learn** – Machine learning models

## What I Did

* Loaded and explored the dataset
* Cleaned missing/inconsistent data
* Visualized trends using plots
* Built and evaluated ML models
* Drew conclusions from the analysis

## Key Findings

This repository represents my hands-on learning journey through
Python's core Data Science libraries. Starting from the basics of
Pandas Series and DataFrames, I progressively worked with real-world
datasets like Uber fares, Google Play Store apps, Student Performance,
Superstore sales, and Yahoo Finance data.

Through this journey, I learned how to clean and preprocess messy
real-world data, handle missing values, duplicates, and DateTime
formats. I built visualizations using Matplotlib and Seaborn to
uncover hidden patterns and trends. The Uber EDA project was a
complete end-to-end analysis — from raw data to meaningful business
insights about fare patterns across time and passenger count.

Every dataset told a different story, and Python gave me the tools
to listen. This is just the beginning — the foundation is set for
diving deeper into Machine Learning and Advanced Data Science! 🚀
